Beautiful Patterns Faucet

This faucet allows you request Beautiful Patterns tokens. To prevent malicious actors from exhausting all available funds or accumulating enough tokens to mount long running spam attacks, requests are tied to Twitter social network accounts. Anyone having a Twitter account may request funds within the permitted limits.

How does this work?

To request funds via Twitter, make a tweet with your Ethereum address pasted into the contents (surrounding text doesn't matter). Copy-paste the tweets URL into the above input box and fire away!
